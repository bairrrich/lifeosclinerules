# 📘 Project Brief: Life OS

## Обзор проекта

**Life OS** — единое приложение для комплексного учета жизненных показателей с принципами offline-first и mobile-first.

## Цели проекта

Создать единое приложение для учета:
- 🍽️ Питания
- 💪 Тренировок
- 💰 Финансов
- 💊 Витаминов
- 💉 Лекарств
- 🌿 Трав
- 💄 Косметики
- 🛒 Продуктов
- 📖 Книг
- 🍳 Рецептов

## Ключевые требования

### Функциональные
- Работа офлайн (offline-first)
- Синхронизация при появлении сети
- Унифицированный интерфейс для всех сущностей
- Поддержка светлой и темной темы
- PWA (Progressive Web App)

### Нефункциональные
- Mobile-first дизайн
- Минималистичный UI
- Нормализованная база данных
- Быстрая загрузка
- Единый стиль интерфейса

## Области учета

### Группа A — Учет (Logs)
Записи с датой, временем и метриками:
- **Питание** — калории, БЖУ, приемы пищи
- **Тренировки** — тип, длительность, интенсивность
- **Финансы** — доходы, расходы, счета

### Группа B — Каталог (Items)
Справочник веществ и продуктов:
- Витамины
- Лекарства
- Травы
- Косметика
- Продукты

### Группа C — Контент (Content)
Контентная информация:
- Книги (статус прочтения, рейтинг)
- Рецепты (ингредиенты, время приготовления)

## Принципы разработки

1. **Одно приложение — один стиль — одна логика — одна структура**
2. Все сущности должны ощущаться частью единой системы
3. Максимальная унификация полей и интерфейсов
4. Минимализм в дизайне
5. Карточная система отображения
